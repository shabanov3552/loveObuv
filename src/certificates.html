Как это работает:

После оформления и отплаты на ваш электронный адрес придет промокод для активации сертификата, который можно использовать при оплате покупок на нашем сайте.

Правила использования:

  • Срок действия: 1 год. Важно использовать сертификат до истечения его срока действия, так как после этого он становится недействительным.
  • Использование: сертификат можно использовать только для покупки товаров в магазине, который его выдал. Он не может быть использован для приобретения товаров в других магазинах или онлайн-магазинах.
  • Невозможность получения наличных: сертификат не переводится в наличные деньги и не может быть использован для оплаты долгов по кредитной карте. Он может быть использован только для покупки товаров в магазине.
  • Покупка товара на меньшую сумму: если товар, который вы хотите купить, стоит меньше, чем сумма на сертификате, то оставшаяся часть денег не возвращается и не может быть использована для будущих покупок.
  • Утеря: если сертификат потерян или украден, он не может быть восстановлен. Сохраняйте его в безопасном месте до использования.
